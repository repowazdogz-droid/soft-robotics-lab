# Cognitive Twin Authoring Contract

This contract defines how Omega authors cognitive twins without simulating identity, personality, or consciousness.

Version: 1.0  
Effective: 2024-12-13

---

## 1. Definition

A cognitive twin is:

### A Model of Decision-Making Behavior
- A cognitive twin is a model of decision-making behavior
- Twins model how decisions are made
- Decision-making behavior modeling is mandatory
- Missing decision-making focus invalidates twin

### Focused on Judgment Patterns, Not Individuals
- A cognitive twin is focused on judgment patterns, not individuals
- Patterns are modeled, not people
- Focusing on judgment patterns is mandatory
- Individual focus is prohibited

### Heuristic-Based, Not Predictive

- A cognitive twin is heuristic-based, not predictive
- Heuristics guide modeling, prediction does not
- Heuristic-based design is mandatory
- Predictive design is prohibited

### A Cognitive Twin Is Not

- A cognitive twin is not a personality replica
- A cognitive twin is not a chatbot
- A cognitive twin is not a consciousness simulation
- A cognitive twin is not an identity substitute

### A Personality Replica
- A cognitive twin is not a personality replica
- Personality replication is not the purpose
- Avoiding personality replica framing is mandatory
- Personality replica framing is prohibited

### A Chatbot
- A cognitive twin is not a chatbot
- Chatbots engage in conversation
- Avoiding chatbot framing is mandatory
- Chatbot framing is prohibited

### A Consciousness Simulation
- A cognitive twin is not a consciousness simulation
- Consciousness simulation is not the purpose
- Avoiding consciousness simulation framing is mandatory
- Consciousness simulation framing is prohibited

### An Identity Substitute

- A cognitive twin is not an identity substitute
- Identity substitution is not the purpose
- Avoiding identity substitute framing is mandatory
- Identity substitute framing is prohibited

- Requiring role-based inputs is mandatory
- Person-based inputs are prohibited

### Limited to Decision-Relevant Signals

- All cognitive inputs must be limited to decision-relevant signals

- Limiting to decision-relevant signals is mandatory

### Implicit Psychological Inference Is Prohibited

- Implicit psychological inference is prohibited
- Psychological inference must be explicit

## Enforcement

Violations of this contract include:
- Cognitive twins presented as personality replicas, chatbots, consciousness simulations, or identity substitutes
- Modeling emotions as internal states, personal memories, desires/intent, or subjective experience
- Implicit psychological inference, person-based inputs, or non-decision-relevant inputs
- Outputting predictions of personal behavior, emotional responses, or intent attribution
- Deterministic cognitive claims, missing uncertainty expression, or overconfidence
- Cognitive twins acting autonomously, making decisions, or replacing human judgment
- Missing drift monitoring, hidden mismatch, or continuing without revalidation
- Missing or incomplete audit trails (modeled heuristics, excluded traits, assumptions, known limitations, usage context)
- Missing override logging

Violations must be:
- Logged as incidents
- Reviewed by system owner
- Corrected immediately
- Documented in change logs

---

## Amendments

Changes to cognitive twin definitions, scope restrictions, input discipline, or output constraints require:
- Impact assessment on cognitive twin authoring and integrity
- Testing with representative twin scenarios
- Approval from system owner
- Version increment
- Documentation in change logs
