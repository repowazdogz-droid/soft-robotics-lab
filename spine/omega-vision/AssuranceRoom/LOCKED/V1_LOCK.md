# AssuranceRoom — V1 LOCK (FROZEN)

Purpose:
A visual-only assurance ladder (inputs → authority → outcomes → overrides → trace).
Human-led. Non-autonomous. Not a simulator. Not a control system. Not certification.

V1 CONTENTS (FROZEN LAYERS)
- InputsLayer: I1–I4 (selectable)
- AuthorityLayer: A1–A4 (selectable)
- OutcomesLayer: S1–S4 (selectable)
- OverridesLayer: OVR1–OVR3 (selectable)
- TraceLayer: TR* (non-selectable)

ALLOWED (V1)
- Camera orbit/pan/zoom
- Layer toggles
- Tap-to-select ONLY I*, A*, S*, OVR* → Inspector shows neutral metadata
- Reset View (camera only)
- Presets that ONLY change Inspector copy

DISALLOWED (V1)
- Any motion, animation, time evolution
- Any "select", "recommend", "best outcome", "choose this"
- Procedures, thresholds, step-by-step instructions
- Claims of compliance/certification
- Editing/dragging objects
- Sliders that imply optimization or decision-making
- Export claiming certification or guarantees

FAILURE SIGNAL
If a user can interpret this as a control system, recommendation engine, or certification tool, V1 has failed.

CHANGE POLICY
- Bug fixes + copy tightening allowed
- Anything affecting behavior, selectability, or semantics requires new lock version (V2)
