TradeoffRoom is a static spatial reasoning surface.

TradeoffRoom visualizes objectives, constraints, feasible regions, and tensions without optimization or recommendation.

It makes tradeoffs visible as objects, and shows:
- objectives
- constraints
- feasible regions
- tradeoff tensions

It does not:
- optimize outcomes
- recommend "best" options
- run simulations
- claim correctness
- select solutions

Human-led. Non-autonomous. Visual reasoning only.
